Abra o projeto no seu editor (vscode ou pycharm) em seguida entre no terminal e coloque o seguinte comando:
'pip install -r requirements.txt'  , para poder importar todas as bibilhotecas usadas no projeto.

//// EXECUTAR O PROJETO ////

Para executar o projeto abra o terminal e digite 'python manage.py runserver' depois entre no navegador e pesquise por
'http://localhost:8000/accounts/login/'

//// Criar usuario e senha ////
Para criar seu login e senha click em criar uma conta , você sera redirecionado a uma nova tela de registro basta preencher suas informações de login e clicar em registrar
Admin padrão = admin
senha padrão = 2611

/// cadastrar pedido ///
para cadastrar um novo pedido basta clicar no botão cadastrar e preencher as informações e no final clicar no botão concluir

/// editar pedido ///

para editar o pedido basta clicar no botão com icone de lapiz

/// excluir pedido ///

para excluir o pedido basta clicar no botão com icone de lixeira e confirmar

/// mudar status do pedido ///

para mudar o status do pedido de pendente para concluido clique no botão de check

/// Relatorio ///

para gerar um relatorio basta clicar em gerar relatorio que fica na nav-bar

/// deslogar ///

para deslogar clique em sair
